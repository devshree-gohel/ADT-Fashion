---

## 🚀 Highlights

- Modern, responsive layout using semantic HTML and handcrafted CSS
- Interactive UI components with lightweight vanilla JavaScript
- Clean file structure ideal for learning, prototyping, and portfolios
- Easy to customize colors, spacing, and assets

## 🔧 Quick start

1. Clone the repo
   ```bash
   git clone https://github.com/devshree-gohel/ADT-Fashion.git
   cd ADT-Fashion
   ```
2. Open `index.html` in your browser (double-click or use `open index.html` on macOS).

---

## 📁 Project structure

- `index.html` — main landing page
- `assets/` — images, fonts, icons
- `css/` — stylesheets (layout, components, responsive rules)
- `js/` — small scripts for UI behavior and interactivity

---

## 🎨 Customize

- Colors: update CSS variables (e.g., `:root`) in the main CSS file.
- Fonts: add fonts in `assets/fonts` or link Google Fonts in the HTML head.
- Layout: adjust Grid/Flex rules in layout CSS to change the page composition.

---

## 🧩 Tech stack

- HTML5
- CSS3 (Flexbox, Grid, custom properties)
- JavaScript (vanilla)
